Программа реализует систему классов Точка, Вектор, Треугольник, Четырехугольник в трехмерной системе координат. Доступны различные варианты сортировки, методы для добавления нового объекта, для удаления объекта, для сдвига на определенный вектор, и для получения симметричных относительно начала координат обЪектов
